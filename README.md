# CRA(ken)

A CRA(ken) is a boilerplate of create react app with a lot of helpers.

## CRA vs CRA(ken)

|                        | CRA(ken)   | CRA   |
| ---------------------- | ---------- | ----- |
| ESLint Configuration   | ✅ **yes** | ❌ no |
| Prettier Configuration | ✅ **yes** | ❌ no |
| Routing                | ✅ **yes** | ❌ no |
